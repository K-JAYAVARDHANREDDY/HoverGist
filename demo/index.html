Features

Everything you need,
nothing you don't

A single JS file, zero dependencies, and a powerful API gateway that handles rate limiting, authentication, and GPT-4 integration for you.

🧠

GPT-4 Powered Summaries

Uses OpenAI's GPT-4 to generate concise, intelligent gists of any section — from product descriptions to complex technical documentation. The model is tuned specifically for web content summarization at 120 words or fewer.

⚡

One-Key Activation

Users press Alt to enter hover mode. The cursor changes, sections glow, and clicking any text-rich element triggers instant summarization. No UI clutter, no popups interrupting the reading flow until requested.

🔑

API Gateway with Rate Limiting

Each developer API key is limited to 200 GPT-4 calls per 24 hours, enforced at the server level. Real-time usage stats are surfaced in the tooltip and the developer config panel. Swap in Redis for distributed rate limiting at scale.

🎨

Fully Customizable

Configure the highlight color, tooltip theme (dark/light/auto), trigger key, max gist word count, and scoped CSS selectors. All via a simple HoverGist.init() call.

📦

Drop-in SDK

A single self-contained JavaScript file — no build step, no dependencies. Include via a <script> tag or import as a CommonJS/AMD module. Works on any website, any framework, any era.

🛡️

Secure by Design

API keys are validated server-side on every request. CORS is enforced per-domain. Text is capped at 4,000 characters before reaching GPT-4 to prevent abuse and minimize token costs.

Why Your Node.js App Is Leaking Memory (And How to Fix It)

Memory leaks in Node.js are notoriously hard to diagnose because V8's garbage collector usually handles cleanup automatically. The most common culprits are event listeners that are never removed, closures holding references to large objects, and global caches that grow without a size limit. Using process.memoryUsage() can surface heap growth trends, while Chrome DevTools' heap snapshot tool lets you compare allocations over time. Tools like clinic.js and 0x can profile the event loop in production. The fix is usually straightforward: WeakMap for caches, explicit removeEventListener calls in cleanup functions, and bounded in-memory stores with LRU eviction.

Tokens, Themes & Components: Building a Scalable Design System in 2025

A robust design system starts with design tokens — primitives like color, spacing, radius, and typography that are defined once and consumed everywhere. Tools like Style Dictionary or Tokens Studio for Figma allow tokens to be exported to CSS custom properties, Tailwind config, iOS Swift constants, and Android XML simultaneously. Components are built on top of semantics, not raw values, meaning a single token change can propagate to every button, card, and input across your entire product. Security

The Modern Developer's Guide to OWASP Top 10 in 2025

The OWASP Top 10 remains the most referenced checklist for web application security. Broken access control still tops the list, followed by cryptographic failures and injection attacks. Modern mitigations include parameterized queries for SQL injection, output encoding for XSS, PKCE for OAuth flows, and strict CSP headers. Server-side request forgery (SSRF) has risen dramatically with the proliferation of cloud metadata APIs — always validate and allowlist outbound HTTP calls from your backend. Insecure design is a new category emphasizing that security must be baked into architecture from day one, not bolted on after. Regular dependency audits with tools like npm audit, Snyk, or Dependabot are now table stakes for any production application.

SDK Integration

Add to your site in 3 lines

Drop the script in, call init, and your users get instant AI summaries on hover.

index.html 
<!-- 1. Include the SDK -->
<script src="https://your-cdn.com/hovergist.sdk.js"></script>

<!-- 2. Initialize -->
<script>
  HoverGist.init({
    apiKey:      'hg_your_key_here',
    backendUrl:  'https://your-backend.com',
    theme:       'dark',          // 'dark' | 'light' | 'auto'
    triggerKey:  'Alt',           // key to activate
    highlightColor: '#6366f1',    // hover ring color
    maxWords:    120,              // gist word limit

    // Optional callbacks
    onGist:      (gist, el) => console.log(gist),
    onActivate:  () => console.log('Active!'),
  });
</script>
